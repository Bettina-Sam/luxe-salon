## 💎 Luxe Salon

An elegant and interactive salon website that blends **luxury beauty branding, modern web design, AI interaction, and playful UI experiences** into a responsive web application.

This project combines **premium UI design, AI chat assistance, camera-based mirror tools, and engaging interactive sections** to create a visually rich and portfolio-ready frontend experience.

---

## ✨ Features

- 💎 Luxury salon inspired web interface  
- 🤖 AI Chat assistant powered by Gemini API  
- 🪞 Mirror camera feature for beauty preview  
- 🎭 Avatar styling and interactive features  
- 🎮 Fun interactive games section  
- 💬 Client testimonials and review section  
- 🌗 Dark mode and light mode support  
- 📱 Fully responsive layout for mobile and desktop  
- ⚡ Fast performance powered by Vite  
- 🧩 Modular React component architecture  

---

## 💇 Main Sections

🏠 **Home**  
Landing page showcasing salon branding, services, testimonials, and interactive UI elements.

💄 **Services**  
Displays salon offerings such as hair styling, skin care, spa services, and beauty treatments.

💬 **AI Chat Assistant**  
Allows users to ask beauty-related questions and receive intelligent responses.

🪞 **Mirror Tool**  
Uses the device camera to allow users to preview themselves and explore styling ideas.

🎭 **Avatar Studio**  
Interactive area where users can experiment with fun avatar styling elements.

🎮 **Games**  
Playful mini-games designed to make the experience engaging and interactive.

📞 **Contact / Booking**  
Provides salon contact information and encourages booking services.
